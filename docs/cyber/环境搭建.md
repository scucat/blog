# 环境搭建
## VMWare 安装 kali linux
[教程](https://blog.csdn.net/weixin_43290709/article/details/117519611)

### 1. 下载kali-linux
[官网](https://www.kali.org/)

Download > Vitrtual Machines > X64

### 2. 解压安装
个人选择解压位置，打开VMware > 打开一个虚拟机 > 选择刚刚下载并解压后的文件`.vmx`

### 3. 启动虚拟机
默认密码 kali/kali
