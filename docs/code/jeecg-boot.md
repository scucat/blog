# JeecgBoot
官网：http://jeecg.com/
源码：https://github.com/jeecgboot/jeecg-boot
Vue3前端代码：https://github.com/jeecgboot/jeecgboot-vue3

## 启动前端

### 安装yarn
```
// yarn查看当前版本
yarn -v
// yarn查看所有版本
npm view yarn versions
// yarn更新到最新版
npm install yarn@latest -g
```

### 运行jeecgboot-vue3
```powershell
yarn
yarn serve

```

## 启动后端

### idea导入jeecgboot右键没有maven选项

选择pom文件，右键 -> "add as Maven Project"
